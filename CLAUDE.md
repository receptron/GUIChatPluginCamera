# CLAUDE.md

## Plugin Overview

Camera photo capture plugin for GUIChat. Allows users to take photos using their device camera.

## Common Guidelines

For standard plugin development guidelines, see:
https://github.com/receptron/GUIChatPluginTemplate/blob/main/CLAUDE.md

## Plugin-Specific Notes

### Features
- Access device camera via browser API
- Capture photos for use in chat
- Preview captured images

### Dependencies
- `gui-chat-protocol`: Core protocol for GUI Chat plugins
